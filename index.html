抓取流程（Playwright 浏览器自动化）
  1. 1解析商品 URL，获取商品 ID
  2. 2Playwright 打开商品详情页（使用源店铺登录态）
  3. 3解析 DOM 提取标题、价格、店铺名
  4. 4拦截 picasso.taobao.com 请求获取图片 CDN
  5. 5拦截 acs.m.taobao.com 接口获取 SKU JSON
  6. 6下载 1:1 主图、3:4 主图、长图、视频
  7. 7解析详情页 iframe 的 descContent HTML

ℹ️ 淘宝开放平台 API 仅支持单店铺管理，无法跨店铺读取商品。 本工具采用 Playwright 浏览器自动化方案，详见「使用说明」Tab。
